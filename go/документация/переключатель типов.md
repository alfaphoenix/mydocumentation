 переключатель типов выглядит следующим образом:
 `"Type switch" — это специальная форма `switch` в Go, которая **не сравнивает значения**, а (сравнивает тип значения внутри интерфейса):`
- То есть мы проверяем: “какой реальный тип хранится внутри интерфейса?”
- Можно проверять **несколько типов подряд** без длинных `if ... else if ...`

```go
switch v := i.(type) {
	case T1:
		...
	case T2, T3:
		...
	default:
		...
}
```

это приведение типов

```go
i.(int)
```
