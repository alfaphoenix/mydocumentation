
## определение переменной

```go
var имя_переменной тип_переменной

var имя_переменной тип_переменной = значение_переменной

var имя_переменной = значение_переменной

имя_переменной := значение_переменной 
// тип переменной выводится исходя из ее значения

имя_переменной_1, имя_переменной_2 := значение_переменной_1, значение_переменной_2

var имя_переменной_1, имя_переменной_2 тип_переменных
// все переменные имеют один тип

var (
	имя_переменной_1 тип_переменной_1
	имя_переменной_2 тип_переменной_2
)

var (
	имя_переменной_1 = значение_переменной_1
	имя_переменной_2 = значение_переменной_2
)
```

- простейшее определение переменной

```go
var hello string
```

-  при объявлении переменной присвоить ей начальное значение
```go
var hello string = "Hello world"
```

- Можно одновременно объявить сразу несколько переменных через запятую:
```go
var a, b, c string
```

- Если мы хотим сразу определить несколько переменных и присвоить им начальные значения, то можно обернуть их в скобки:
```go
var (
	name string = "Tom"
	age int = 27
)
```

- ### Краткое определение переменной

```go
// имя_переменной := значение
name := "Tom"
```

Также этот способ можно использовать для объявления и инициализации нескольких переменных в одной строке:

```go
x, y := 5, 11 
```
## Примечание 

- Имя может состоит из алфавитных и цифровых символов и символа подчеркивания. Первым символом должен быть либо алфавитный символ, либо символ подчеркивания. При этом имена не должны представлять одно из ключевых слов:
```go
break, case, chan, const, continue, default, defer, else, 
fallthrough, for, func, go, goto, if, import, interface, 
map, package, range, return, select, struct, switch, type, var
```

- Go - регистр зависимый язык, то есть переменные с именами "hello" и "Hello" будут представлять разные переменные:
```go
var hello string

hello = "Hello world"

fmt.Println(Hello)  // ! Ошибка переменной Hello нет, есть переменная hello
```
