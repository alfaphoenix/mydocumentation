
Рекурсивная функция представляет такую функцию, которая вызывает саму себя. Рекурсивные функции представляют мощный инструмент для обработки рекурсивных структур данных, например, различных деревьев.

Рассмотрим простейший, хотя в некоторой степени искусственный пример с вычислением суммы чисел:

```go 
package main
 
import "fmt"
 
func sum(n uint) uint{
 
    if n == 1{
        return n
    }
    return n + sum(n - 1)
}
func main() {
     
    fmt.Println(sum(4)) // 10
    fmt.Println(sum(5)) // 15
    fmt.Println(sum(6)) // 21
}
```

Здесь функция sum получает некоторое число и вычисляет сумму чисел от 0 до этого числа. Полученный результат возвращается из функции.
