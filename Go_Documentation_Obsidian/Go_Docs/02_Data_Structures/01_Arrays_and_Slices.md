–û–ø–µ—Ä–∞—Ç–æ—Ä —Å—Ä–µ–∑–∞ `[low:high]` –≤—ã–±–∏—Ä–∞–µ—Ç –ø–æ–ª—É–æ—Ç–∫—Ä—ã—Ç—ã–π –¥–∏–∞–ø–∞–∑–æ–Ω —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –≤–∫–ª—é—á–∞—è `low`, –Ω–æ –∏—Å–∫–ª—é—á–∞—è `high`.

### –î–ª–∏–Ω–∞ –∏ –ï–º–∫–æ—Å—Ç—å

–°—Ä–µ–∑—ã –∏–º–µ—é—Ç **–¥–ª–∏–Ω—É** (`len`) –∏ **–µ–º–∫–æ—Å—Ç—å** (`cap`).
