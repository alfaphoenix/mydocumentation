## Операторы присваивания
Операторы присваивания устанавливают значение переменной. Их можно использовать для присвоения значения новой переменной или изменения значения переменной путем вычисления любого определенного выражения. Эти операторы сначала вычисляют выражение, заданное справа от оператора присваивания, а затем присваивают конечный результат переменной, записанной слева от оператора присваивания. В языке Go есть несколько операторов присваивания:

- =
    Присваивает левому операнду значение правого операнда.
- :=
    Присваивает левому операнду, которая ранее не был определен, значение правого операнда .
- +=
    Присваивает левому операнду сумму значений левого и правого операнда.
- -=
    Присваивает левому операнду разность значений левого и правого операнда.
- `*=` 
    Присваивает левому операнду произведение значений левого и правого операнда
- /=
    Присваивает левому операнду результат деления значений левого операнда на правый.
- %=
    Присваивает левому операнду результат остатка от деления значений левого операнда на правый.
- <<=
    Присваивает левому операнду результат сдвига влево значения левого операнда на значение правого операнда
- >>=
    Присваивает левому операнду результат сдвига вправо значения левого операнда на значение правого операнда.
- &=
    Присваивает левому операнду результат побитового умножения левого и правого операндов.
- ^=
    Присваивает левому операнду результат операции XOR между левым и правым операндами.
- |=
    Присваивает левому операнду результат побитового сложения левого и правого операндов.

Пример использования операторов:

```go
package main
import "fmt"

func main() {

    var a int = 10  // простое присвоение
    
    b := 5
    a += b
    fmt.Println("a += b:", a) // a += b: 15
    
    a -= b
    fmt.Println("a -= b:", a)   // a -= b: 10
    
    a *= 10
    fmt.Println("a *= 10:", a)  // a *= 10: 100
    
    a /= b
    fmt.Println("a /= b:", a)   // a /= b: 20
    
    a %= 15 
    fmt.Println("a %= 15:", a)  // a %= 15: 5
    
    a <<= 2
    fmt.Println("a <<= 2:"``, a)// a <<= 2: 20
    
    a >>= 1  // 10100 >>= 1 = 1010
    fmt.Println("a >>= 1:"``, a)// a >>= 1: 10
    
    a &= 8    // 10100 &=1 = 1000
    fmt.Println("a &= 8:"``, a)   // a &= 8: 8
    
    a ^= 10     // 1000 ^= 1010 = 0010
    fmt.Println("a ^= 10:", a)  // a ^= 10: 2
    
    a |= 5     // 010 |= 101 = 111
    fmt.Println("a |= 5:", a)   // a |= 5: 7
}
```

## Приоритет операторов

Приоритет оператора (Operator precedence) определяет приоритет выполнения каждого оператора, чтобы выражение можно было правильно оценить. Операторы с высоким приоритетом оцениваются первыми при оценке выражения. Ассоциативность возникает, когда у нас есть два или более выражения, имеющие один и тот же оператор или операторы с одинаковым приоритетом. В этом случае выражение оценивается либо слева направо, либо справа налево в соответствии с их правилами ассоциативности. В языке Go приоритет и ассоциативность операторов можно описать с помощью следующей таблицы:

| Категория          | Операторы                          | Ассоциативность |
| ------------------ | ---------------------------------- | --------------- |
| Постфиксные        | () [] -> . ++ --                   | Слева направо   |
| Унарные            | + - ! ~ ++ - - (type)* &sizeof     | Справа налево   |
| Умножение/Деление  | * / %                              | Слева направо   |
| Сложение/Вычитание | + -                                | Слева направо   |
| Сдвиг              | << >>                              | Слева направо   |
| Сравнение          | < <= > >=                          | Слева направо   |
| Equality           | == !=                              | Слева направо   |
| Поразрядное AND    | &                                  | Слева направо   |
| Поразрядное XOR    | ^                                  | Слева направо   |
| Поразрядное OR     | \|                                 | Слева направо   |
| Логическое AND     | &&                                 | Слева направо   |
| Логическое OR      | \|\|                               | Слева направо   |
| ПРисвоение         | = += -= *= /= %= >>= <<= &= ^= \|= | Справа налево   |
