## hello.go

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}
```

запуск 

```cmd
$ go run hello.go
Hello, world!
```

или попробуй  [Go repl](https://repl.it/languages/go)

## 3. Functions

```go 
package main

import "fmt"

// The entry point of the programs
func main() {
    fmt.Println("Hello world!")
    say("Hello Go!")
}

func say(message string) {
    fmt.Println("You said: ", message)
}
```
