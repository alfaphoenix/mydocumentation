- ## Создаем go. mod 

  Пример
  
```go
module myapp // название модуля

go 1.22 
```

- ## Создание библиотеки 
1. Создаем папку для библиотеки (имя папки должно совпадать с именным пакета)
```go 
package shizamath

// папка у нас тоже должен быть shizamath
```
2. Все нужные функции в библиотеке с большой буквы 

- ## Импорт 

```go 
import (
    "myapp/shizamath"
)
```
