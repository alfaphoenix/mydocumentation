# Arguments & Returns

```c++

int add(int a, int b) {
    return a + b;  
}

int main() {
    std::cout << add(10, 20); 
}
```

# Overloading

```c++
void fun(string a, string b) {
    std::cout << a + " " + b;
}
void fun(string a) {
    std::cout << a;
}
void fun(int a) {
    std::cout << a;
}
```

# Built-in Functions

```c++
#include <iostream>
#include <cmath> // import library
 
int main() {
    // sqrt() is from cmath
    std::cout << sqrt(9);
}
```
